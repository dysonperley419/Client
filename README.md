# Flicker

WIP Spacebar compatible (themed as comfy lavender) client that also works with Oldcord instances.

## Implemented Features

 - Messaging, of which:
     * Creating messages, fetching messages, and uploading attachment media - Including ghost messages for currently sending messages/attachments
       
 - Guilds, of which:
    * Creation, joining
    * Member list & User Popout Profiles
  
- Channels, of which:
    * Voice state tracking, and vc member list
    * DM Channels (Groups not tested yet)

- WEBRTC Voice (WIP), of which:
    * Can join an oldcord instance VC locally on firefox/waterfox can speak/hear from Chrome and/to other Firefox
    * Chrome cannot speak yet for some odd reason


## Setup

Run `npm install` and `npm run dev`.

Works well with the latest LTS version of Node (Node 24), for ideal security practice, we aim for latest LTS compatibility.

## HELP WANTED

Help is actively wanted to implement more features into the client & clean/organize existing files.

The end goal for Flicker is to be a suitable daily driver which is essentially bulletproof to takedowns (custom designed/open-licensed icons, and free-to-use fonts for commercial and non commerical purposes, etc).
